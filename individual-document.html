Step-by-Step Process

  1. Start of the process steps.
  2. Next step with screenshot link.

    View screenshot

  3. Select the tab you wish to personsalize (i.e., My Work, Links, Queries, Reports/Processes).

    4. Personalize Screenshot

  4. For the My Work tab, options include:
    • Select if My Work Grouplet is collapsed on start.
    • Select if a group is collapsed on start.
    • Select preferred starting page.
    • Select if you would like counts for each category to show.
    • If options are not changed, the system default remains.
  5. For the Links, Queries, and Report/Processes tab:
    Note: The steps below highlight the Links tab. Follow similar steps to personalize the Queries tab and Report/Processes tab.
    1. Click Add Group.
    2. Enter a Group Label.
    3. Click Define.
    4. Select URL from the Link Type drop-down list.
    5. Enter URLID.
    6. If applicable, enter a Link Label.
    7. Click Done.
    8. Repeat process if adding multiple groups.
  6. Click Reorder Group to arrange the order.
    • Drag and drop to arrange the new order
  7. Click Done.
